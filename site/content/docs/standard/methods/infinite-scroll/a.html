Import

Importing components depends on how your application works. If you intend to use the TW elements ES format, you must first import the component and then initialize it with the initTE method. If you are going to use the UMD format, just import the tw-elements package.

{{< twsnippet/no-demo id="api-example1" >}}
{{< twsnippet/wrapper "javascript,umd" >}} {{< twsnippet/code active=true lang="js" >}} import { InfiniteScroll, initTE } from "tw-elements"; initTE({ InfiniteScroll }); {{< /twsnippet/code >}} {{< twsnippet/code lang="js" >}} import "tw-elements"; {{< /twsnippet/code >}} {{< /twsnippet/wrapper >}}

Usage

Via data attributes

Simply add data-te-infinite-scroll-init attribute to the element to initialize the Infinite Scroll component. For ES format, you must first import and call the initTE method.

{{< twsnippet/no-demo id="api-example2" >}}
{{< twsnippet/wrapper "HTML" >}} {{< twsnippet/code active=true lang="html" >}}
    ...
{{< /twsnippet/code >}} {{< /twsnippet/wrapper >}}

Via JavaScript

{{< twsnippet/no-demo id="api-example3" >}}
{{< twsnippet/wrapper "javascript,umd" >}} {{< twsnippet/code active=true lang="js" >}} const myInfiniteScroll = new InfiniteScroll(document.getElementById('infiniteScroll'), options); {{< /twsnippet/code >}} {{< twsnippet/code lang="js" >}} const myInfiniteScroll = new te.InfiniteScroll(document.getElementById('infiniteScroll'), options) {{< /twsnippet/code >}} {{< /twsnippet/wrapper >}}

Options

Options can be passed via data attributes or JavaScript. For data attributes, append the option name to data-te-, as in data-te-infinite-direction="".

Name Type Default Description
infiniteDirection String 'y' Defines an example scroll direction.

Methods

Method Description Example
dispose Manually deletes an instance of infinite scroll. myInfiniteScroll.dispose()
getInstance Static method which allows you to get the infinite scroll instance associated to a DOM element. InfiniteScroll.getInstance(myInfiniteScrollEl)
getOrCreateInstance Static method which returns the infinite scroll instance associated to a DOM element or create a new one in case it wasn't initialized. InfiniteScroll.getOrCreateInstance(myInfiniteScrollEl)
{{< twsnippet/no-demo id="api-example4" >}}
{{< twsnippet/wrapper "javascript,umd" >}} {{< twsnippet/code active=true lang="js" >}} const myInfiniteScrollEl = document.getElementById('myInfiniteScroll'); const infiniteScroll = new InfiniteScroll(myInfiniteScrollEl); infiniteScroll.dispose(); {{< /twsnippet/code >}} {{< twsnippet/code lang="js" >}} const myInfiniteScrollEl = document.getElementById('myInfiniteScroll'); const infiniteScroll = new te.InfiniteScroll(myInfiniteScrollEl); infiniteScroll.dispose(); {{< /twsnippet/code >}} {{< /twsnippet/wrapper >}}

Events

Event type Description
complete.te.infiniteScroll This event fires immediately after scrolling to the end of the container.
{{< twsnippet/no-demo id="api-example5" >}}
{{< twsnippet/wrapper "JavaScript" >}} {{< twsnippet/code active=true lang="js" >}} const infiniteScrollElement = document.getElementById('element'); infiniteScrollElement.addEventListener('complete.te.infiniteScroll', (e) => { // do something... }); {{< /twsnippet/code >}} {{< /twsnippet/wrapper >}}
