Import

Importing components depends on how your application works. If you intend to use the Tailwind Elements ES format, you must first import the component and then initialize it with the initTE method. If you are going to use the UMD format, just import the tw-elements package.

{{< twsnippet/no-demo id="api-example1" >}}
{{< twsnippet/wrapper "javascript,umd" >}} {{< twsnippet/code active=true lang="js" >}} import { SmoothScroll, initTE } from "tw-elements"; initTE({ SmoothScroll }); {{< /twsnippet/code >}} {{< twsnippet/code lang="js" >}} import "tw-elements"; {{< /twsnippet/code >}} {{< /twsnippet/wrapper >}}

Usage

Via data attributes

Simply add data-te-smooth-scroll-init attribute to the anchor tag to initialize the Smooth Scroll component. For ES format, you must first import and call the initTE method.

{{< twsnippet/no-demo id="api-example2" >}}
{{< twsnippet/wrapper "HTML" >}} {{< twsnippet/code active=true lang="html" >}} Smooth Scroll to #section-1 {{< /twsnippet/code >}} {{< /twsnippet/wrapper >}}

Via JavaScript

{{< twsnippet/no-demo id="api-example3" >}}
{{< twsnippet/wrapper "javascript,umd" >}} {{< twsnippet/code active=true lang="js" >}} const mySmoothScroll = new SmoothScroll(document.getElementById('smoothScroll'), options); {{< /twsnippet/code >}} {{< twsnippet/code lang="js" >}} const mySmoothScroll = new te.SmoothScroll(document.getElementById('smoothScroll'), options); {{< /twsnippet/code >}} {{< /twsnippet/wrapper >}}

Options

Options can be passed via data attributes or JavaScript. For data attributes, append the option name to data-te-, as in data-te-container="".

Name Type Default Description
container String 'body' Changes container of scrolling element with overflow-scroll class.
duration Number 500 Changes duration of smooth scroll.
easing String 'linear' Changes motion type of smooth scroll, available easings: 'linear', 'easeInQuad', 'easeInCubic', 'easeInQuart', 'easeInQuint', 'easeInOutQuad' 'easeInOutCubic', 'easeInOutQuart', 'easeInOutQuint', 'easeOutQuad', 'easeOutCubic', 'easeOutQuart', 'easeOutQuint'.
offset Number 0 Changes offset from element.

Methods

Method Description Example
cancelScroll Manually cancels smooth scroll. smoothScroll.cancelScroll();
dispose Manually disposes an instance. smoothScroll.dispose();
getInstance Static method which allows you to get the smooth scroll instance associated to a DOM element. SmoothScroll.getInstance(smoothScrollEl)
getOrCreateInstance Static method which returns the smooth scroll instance associated to a DOM element or create a new one in case it wasn't initialized. SmoothScroll.getOrCreateInstance(smoothScrollEl)
{{< twsnippet/no-demo id="api-example4" >}}
{{< twsnippet/wrapper "javascript,umd" >}} {{< twsnippet/code active=true lang="js" >}} const mySmoothScrollEl = document.getElementById('mySmoothScroll'); const smoothScroll = new SmoothScroll(mySmoothScrollEl); setTimeout(() => { smoothScroll.cancelScroll(); }, 100); {{< /twsnippet/code >}} {{< twsnippet/code lang="js" >}} const mySmoothScrollEl = document.getElementById('mySmoothScroll'); const smoothScroll = new te.SmoothScroll(mySmoothScrollEl); setTimeout(() => { smoothScroll.cancelScroll(); }, 100); {{< /twsnippet/code >}} {{< /twsnippet/wrapper >}}

Events

Event type Description
scrollCancel.te.smoothScroll This event fires immediately after canceling smooth scrolling.
scrollEnd.te.smoothScroll This event fires immediately when the smooth scroll ends.
scrollStart.te.smoothScroll This event fires immediately when the smooth scroll is starting.
