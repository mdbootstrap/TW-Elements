Select with search inside a modal

Due to a focus trap in modals, it is not possible to focus the outer elements (like select dropdown). You can use select data-te-select-container option to resolve this problem.

The data-te-select-container accepts selector of the element inside of wich select dropdown will be rendered. In this case, the selector should point to the modal container (the element with attribute data-te-modal-init). It is important to use a unique selector to assign select to a specific modal.

Validation

Set data-te-select-validation option to true to enable component validation. The selectValidFeedback and selectInvalidFeedback options allow to modify the validation messages.

{{< twsnippet/demo id="example16">}}
{{< /twsnippet/demo >}}
{{< twsnippet/wrapper "HTML,JAVASCRIPT" >}} {{< twsnippet/code active=true lang="HTML" >}}
{{< /twsnippet/code >}} {{< twsnippet/code lang="js" >}} (() => { "use strict"; // Fetch all the forms we want to apply custom Bootstrap validation styles to const forms = document.querySelectorAll(".needs-validation"); // Loop over them and prevent submission Array.prototype.slice.call(forms).forEach((form) => { form.addEventListener( "submit", (event) => { event.preventDefault(); event.stopPropagation(); form.setAttribute("data-te-was-validated", ""); }, false ); }); })(); {{< /twsnippet/code >}} {{< /twsnippet/wrapper >}}

Set value

The setValue method allows to programatically set the component selections.

Single selection

Add single value string to the arguments list to correctly select option with corresponding value in single selection mode.

{{< twsnippet/demo id="example17">}}
{{< /twsnippet/demo >}}
{{< twsnippet/wrapper "HTML,JAVASCRIPT" >}} {{< twsnippet/code active=true lang="HTML" >}}
{{< /twsnippet/code >}} {{< twsnippet/code lang="js" >}} const singleSelect = document.querySelector("#singleSelection"); const singleSelectInstance = te.Select.getInstance(singleSelect); singleSelectInstance.setValue("4"); {{< /twsnippet/code >}} {{< /twsnippet/wrapper >}}

Multi selection

Add array of string values to the arguments list to correctly select option with corresponding value in multi selection mode.

{{< twsnippet/demo id="example18">}}
{{< /twsnippet/demo >}}

{{< /twsnippet/code >}} {{< twsnippet/code lang="js" >}} const multiSelect = document.querySelector("#multiSelection"); const multiSelectInstance = te.Select.getInstance(multiSelect); multiSelectInstance.setValue(["3", "4", "5"]); {{< /twsnippet/code >}} {{< /twsnippet/wrapper >}}

Select with toggle element

If you want to toggle Select via button, you have to add data-te-select-toggle attribute to this button with ID of the Select element.

{{< twsnippet/demo id="example19">}}
{{< /twsnippet/demo >}}
{{< twsnippet/wrapper "HTML,JAVASCRIPT" >}} {{< twsnippet/code active=true lang="HTML" >}}
{{< /twsnippet/code >}} {{< twsnippet/code lang="js" >}} const selectEl = document.getElementById("mySelect"); const select = te.Select.getInstance(selectEl); document.getElementById("toggleMySelect").onclick = function () { select.open(); }; {{< /twsnippet/code >}} {{< /twsnippet/wrapper >}}
