Import

Importing components depends on how your application works. If you intend to use the TW elements ES format, you must first import the component and then initialize it with the initTE method. If you are going to use the UMD format, just import the tw-elements package.

{{< twsnippet/no-demo id="api-example41" >}}
{{< twsnippet/wrapper "javascript,umd" >}} {{< twsnippet/code active=true lang="js" >}} import { Button, initTE } from "tw-elements"; initTE({ Button }); {{< /twsnippet/code >}} {{< twsnippet/code lang="js" >}} import "tw-elements"; {{< /twsnippet/code >}} {{< /twsnippet/wrapper >}}

Via data attributes

For all buttons except fixed buttons

Elements with the data-te-toggle="button" attribute will be automatically initialized - you can set all available options with data attributes. For ES format, you must first import and call the initTE method.

Via JavaScript

You can access an instance from your Javascript code, by using the getInstance static method of the Button class - it allows making use of all the public methods listed in the Methods section.

{{< twsnippet/no-demo id="api-example4" >}}
{{< twsnippet/wrapper "javascript,umd" >}} {{< twsnippet/code active=true lang="js" >}} const exampleEl = document.querySelector('example'); const myButton = new Button(exampleEl); {{< /twsnippet/code >}} {{< twsnippet/code lang="js" >}} const exampleEl = document.querySelector('example'); const myButton = new te.Button(exampleEl); {{< /twsnippet/code >}} {{< /twsnippet/wrapper >}}

Methods

Method Description Example
dispose Destroys an element's button. instance.dispose()
getInstance Static method which allows you to get the button instance associated to a DOM element. Button.getInstance(button)
getOrCreateInstance Static method which returns an alert instance associated to a DOM element or create a new one in case it wasn't initialized. Button.getOrCreateInstance(button)
toggle Toggles push state. Gives the button the appearance that it has been activated. instance.toggle()
show Manually opens fixed button list. instance.show()
hide Manually hides fixed button list. instance.hide()
{{< twsnippet/no-demo id="api-example5" >}}
{{< twsnippet/wrapper "javascript,umd" >}} {{< twsnippet/code active=true lang="js" >}} const buttons = document.querySelectorAll('button'); buttons.forEach((button) => { const button = new Button(button); button.toggle(); }); {{< /twsnippet/code >}} {{< twsnippet/code lang="js" >}} const buttons = document.querySelectorAll('button'); buttons.forEach((button) => { const button = new te.Button(button); button.toggle(); }); {{< /twsnippet/code >}} {{< /twsnippet/wrapper >}}

Events

Event type Description
show.te.button This event fires immediately when the fixed button show method has been called.
shown.te.button This event is fired when a fixed button list has been made visible to the user (will wait for CSS transitions to complete).
hide.te.button This event is fired immediately when the fixed button hide method has been called.
hidden.te.button This event is fired when a fixed button list has been hidden from the user (will wait for CSS transitions to complete).
