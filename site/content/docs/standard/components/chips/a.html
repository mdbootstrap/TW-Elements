Import

Importing components depends on how your application works. If you intend to use the TW elements ES format, you must first import the component and then initialize it with the initTE method. If you are going to use the UMD format, just import the tw-elements package.

{{< twsnippet/no-demo id="api-example41" >}}
{{< twsnippet/wrapper "javascript,umd" >}} {{< twsnippet/code active=true lang="js" >}} import { Chip, ChipsInput, initTE } from "tw-elements"; initTE({ Chip, ChipsInput }); {{< /twsnippet/code >}} {{< twsnippet/code lang="js" >}} import "tw-elements"; {{< /twsnippet/code >}} {{< /twsnippet/wrapper >}}

Via JavaScript

You can access an instance from your Javascript code, by using the getInstance static method of the ChipsInput class - it allows making use of all the public methods listed in the Methods section.

{{< twsnippet/no-demo id="api-example4" >}}
{{< twsnippet/wrapper "javascript,umd" >}} {{< twsnippet/code active=true lang="js" >}} const chipsInput = new ChipsInput( document.getElementById("chipsInputId"), options, classes ); const chip = new Chip( document.getElementById("chipId"), options, classes ); {{< /twsnippet/code >}} {{< twsnippet/code lang="js" >}} const chipsInput = new te.ChipsInput( document.getElementById("chipsInputId"), options, classes ); const chip = new te.Chip( document.getElementById("chipId"), options, classes ); {{< /twsnippet/code >}} {{< /twsnippet/wrapper >}}

Options

Options can be passed via data attributes or JavaScript. For data attributes, append the option name to data-te-, as in data-te-editable="true".

Name Type Default Description
inputID string '' Change input ID
parentSelector '' '' Set parent selector to show chips.
initialValues Array [{ tag: 'init1' }, {tag: 'init2' }] Add custom tags on init.
editable Boolean false Change editable options to chips in data-te-chips-placeholder or data-te-chips-initial
labelText String Example label Change label
inputClasses Object {} Allows to pass the custom classes to the Input component. Used in ChipsInput.
inputOptions Object {} Allows to pass the options to the Input component. Used in ChipsInput.

Methods

Method Description Example
dispose Destroy chips myChips.dispose()
getInstance Static method which allows you to get the chips instance associated to a DOM element. ChipsInput.getInstance(myChips)
getOrCreateInstance Static method which returns the chips instance associated to a DOM element or create a new one in case it wasn't initialized. ChipsInput.getOrCreateInstance(myChips)

Events

Event type Description
add.te.chips This event fires start if you add chip to data-te-chips-placeholder or data-te-chips-initial
arrowDown.te.chips This event fires start if you click arrow down in the data-te-chips-placeholder or data-te-chips-initial
arrowLeft.te.chips This event fires start if you click arrow left in the data-te-chips-placeholder or data-te-chips-initial
arrowRight.te.chips This event fires start if you click arrow right in the data-te-chips-placeholder or data-te-chips-initial
arrowUp.te.chips This event fires start if you click arrow up in the data-te-chips-placeholder or data-te-chips-initial
delete.te.chips This event fires start if you click delete button/backspace in the data-te-chips-placeholder or data-te-chips-initial
