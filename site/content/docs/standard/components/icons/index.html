Fixed Width Icons

Set one or more icons to the same fixed width. This is great to use when varying icon widths (e.g. a tall but skinny icon atop a wide but short icon) would throw off vertical alignment. There is background color on the icon so you can see the fixed width and also bumped up the font-size of the parent element.
