Import

Importing components depends on how your application works. If you intend to use the TW elements ES format, you must first import the component and then initialize it with the initTE method. If you are going to use the UMD format, just import the tw-elements package.

{{< twsnippet/no-demo id="api-example41" >}}
{{< twsnippet/wrapper "javascript,umd" >}} {{< twsnippet/code active=true lang="js" >}} import { Stepper, initTE } from "tw-elements"; initTE({ Stepper }); {{< /twsnippet/code >}} {{< twsnippet/code lang="js" >}} import "tw-elements"; {{< /twsnippet/code >}} {{< /twsnippet/wrapper >}}

Usage

Via data attributes

Elements with the data-te-stepper-init attribute will be automatically initialized - you can set all available options with data attributes. For ES format, you must first import and call the initTE method.

{{< twsnippet/no-demo id="api-example1" >}}
{{< twsnippet/wrapper "HTML" >}} {{< twsnippet/code active=true lang="html" >}}
    ...
{{< /twsnippet/code >}} {{< /twsnippet/wrapper >}}

Via JavaScript

You can access an instance from your Javascript code, by using the getInstance static method of the Stepper class - it allows making use of all the public methods listed in the Methods section.

{{< twsnippet/no-demo id="api-example4" >}}
{{< twsnippet/wrapper "javascript,umd" >}} {{< twsnippet/code active=true lang="js" >}} const stepper = new Stepper(document.getElementById('stepper')); {{< /twsnippet/code >}} {{< twsnippet/code lang="js" >}} const stepper = new te.Stepper(document.getElementById('stepper')); {{< /twsnippet/code >}} {{< /twsnippet/wrapper >}}

Options

Options can be passed via data attributes or JavaScript. For data attributes, append the option name to data-te-, as in data-te-stepper-type="vertical".

Name Type Default Description
stepperAnimationDuration number 800 Set the animation duration in milliseconds. Should be the same as the duration provided by animation classes.
stepperType string 'horizontal' Set stepper view.
stepperLinear boolean false Set to true to use the linear stepper.
stepperNoEditable boolean false Set to true to block editing of the completed step.
slideInLeftAnimation string animate-[slide-in-left_0.8s_both] Sets the animation class for the slide in left animation.
slideOutLeftAnimation string animate-[slide-out-left_0.8s_both] Sets the animation class for the slide out left animation.
slideInRightAnimation string animate-[slide-in-right_0.8s_both] Sets the animation class for the slide in right animation.
slideOutRightAnimation string animate-[slide-out-right_0.8s_both] Sets the animation class for the slide out right animation.

Methods

Method Description Example
changeStep Switch to the step given as the parameter. Stepper.changeStep()
getInstance Static method which allows you to get the stepper instance associated to a DOM element. Stepper.getInstance(myStepperEl)
getOrCreateInstance Static method which returns the stepper instance associated to a DOM element or create a new one in case it wasn't initialized. Stepper.getOrCreateInstance(myStepperEl)
nextStep Switch to the next step. stepper.nextStep()
previousStep Switch to the previous step. stepper.previousStep()
{{< twsnippet/no-demo id="api-example5" >}}
{{< twsnippet/wrapper "javascript,umd" >}} {{< twsnippet/code active=true lang="js" >}} const myStepperEl = document.getElementById('stepper'); const stepper = new Stepper(myStepperEl); stepper.nextStep(); {{< /twsnippet/code >}} {{< twsnippet/code lang="js" >}} const myStepperEl = document.getElementById('stepper'); const stepper = new te.Stepper(myStepperEl); stepper.nextStep(); {{< /twsnippet/code >}} {{< /twsnippet/wrapper >}}
