Basic example

For light design and bright compositions use standard shadows. To apply a shadow to an element simply add one of the following classes to it.

{{< twsnippet/demo id="example1">}}
.shadow-none removes shadows
.shadow-sm
.shadow
.shadow-md
.shadow-lg
.shadow-xl
