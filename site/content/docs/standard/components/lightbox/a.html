Import

Importing components depends on how your application works. If you intend to use the TW elements ES format, you must first import the component and then initialize it with the initTE method. If you are going to use the UMD format, just import the tw-elements package.

{{< twsnippet/no-demo id="api-example1" >}}
{{< twsnippet/wrapper "javascript,umd" >}} {{< twsnippet/code active=true lang="js" >}} import { Lightbox, initTE } from "tw-elements"; initTE({ Lightbox }); {{< /twsnippet/code >}} {{< twsnippet/code lang="js" >}} import "tw-elements"; {{< /twsnippet/code >}} {{< /twsnippet/wrapper >}}

Usage

Via data attributes

Using the Lightbox component doesn't require any additional JavaScript code - simply add a div wrapper with a data-te-lightbox-init attribute and use other data attributes to set all options. For ES format, you must first import and call the initTE method.

Methods

Method Description Example
close Closes the gallery. myLightbox.close()
dispose Removes the Lightbox instance. myLightbox.dispose()
getOrCreateInstance Static method which returns the lightbox instance associated to a DOM element or create a new one in case it wasn't initialized. myLightbox.getOrCreateInstance(element)
getInstance Static method which allows to get the lightbox instance associated with a DOM element. myLightbox.getInstance(element)
open Opens the gallery and activates image set by the target key. Default is the first image. myLightbox.open(1)
reset Resets changed parameters such as position, zoom or fullscreen. myLightbox.reset()
slide Slides an image in the gallery. Available options: left, right, first, last. Dafault is right direction. myLightbox.slide('last')
toggleFullscreen Toggles fullscreen. myLightbox.toggleFullscreen()
zoomIn Zooms in active image by one level. myLightbox.zoomIn()
zoomout Zooms out active image by one level. myLightbox.zoomOut()

Events

Event type Description
open.te.lightbox Emitted when a lightbox starts opening.
opened.te.lightbox Emitted when a lightbox is opened.
slide.te.lightbox Emitted when a lightbox starts sliding.
slided.te.lightbox Emitted after an image slide.
zoomIn.te.lightbox Emitted when an image starts zooming in.
zoomedIn.te.lightbox Emitted after an image is zoomed in.
zoomOut.te.lightbox Emitted when an image starts zooming out.
zoomedOut.te.lightbox Emitted after an image is zoomed out.
close.te.lightbox Emitted when a lightbox starts closing.
closed.te.lightbox Emitted when a lightbox is closed.
