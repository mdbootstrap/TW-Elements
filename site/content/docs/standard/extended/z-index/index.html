Basic example

To place elements on top of others, use the .z-{value} class.

{{< twsnippet/demo id="example1" >}}
z-40
z-30
z-20
z-10
z-0
{{< /twsnippet/demo >}}
{{< twsnippet/wrapper "HTML" >}} {{< twsnippet/code active=true lang="HTML" >}}
z-40
z-30
z-20
z-10
z-0
{{< /twsnippet/code >}} {{< /twsnippet/wrapper >}}

To use a negative z-index value, prefix the class name with a dash to convert it to a negative value.

{{< twsnippet/wrapper "HTML" >}} {{< twsnippet/code active=true lang="HTML" >}}
-z-40
{{< /twsnippet/code >}} {{< /twsnippet/wrapper >}}
