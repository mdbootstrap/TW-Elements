Basic example

Breadcrumb components are an important component in any website or application that can be used to indicate the current position of a page in a hierarchical page structure.

Use the following breadcrumb example to show the hierarchical structure of pages.

{{< twsnippet/demo id="example1">}}
  1. Home

  1. Home
  2. /
  3. Library

  1. Home
  2. /
  3. Library
  4. /
  5. Data
{{< /twsnippet/demo >}}
{{< twsnippet/wrapper "HTML" >}} {{< twsnippet/code active=true lang="HTML" >}}
  1. Home

  1. Home
  2. /
  3. Library

  1. Home
  2. /
  3. Library
  4. /
  5. Data
{{< /twsnippet/code>}} {{< /twsnippet/wrapper>}}
